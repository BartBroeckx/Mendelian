Mendelian is an R package that can be used to filter sequencing variants. It enables filtering for both dominant and recessive traits, with a wide variety of options (including reduced penetrance and detectance). 

This package can be installed from github using: 

```
## Loading required package: devtools
## Downloading github repo BartBroeckx/Mendelian@master
## Installing Mendelian
## "C:/PROGRA~1/R/R-31~1.2/bin/i386/R" --vanilla CMD build  \
##   "C:\Users\Bart\AppData\Local\Temp\Rtmpq4FzNM\devtools13844ffe6994\BartBroeckx-Mendelian-958e034"  \
##   --no-resave-data --no-manual 
## 
## "C:/PROGRA~1/R/R-31~1.2/bin/i386/R" --vanilla CMD INSTALL  \
##   "C:/Users/Bart/AppData/Local/Temp/Rtmpq4FzNM/Mendelian_0.1.tar.gz"  \
##   --library="C:/Users/Bart/Documents/R/win-library/3.1" --install-tests
```

Detailed examples are available in the vignette.
